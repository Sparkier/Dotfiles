# Brew
The Brewfile located in this folder can be executed with:
```zsh
brew bundle --file=Brewfile --verbose
```

It installs the following software:

1. Brew
- python3
- ffmpeg
- node
- ffmpeg
- git-flow-avh
- git
- fzf
- cmake
- ffmpeg
- zsh
- vim
2. Brew Cask
- iterm2
- spectacle
- dash
- fork
- mactex
- slack
- visual-studio-code
- whatsapp
- font-hack-nerd-font
